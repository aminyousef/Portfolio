Modular Robot

A self Assembly Robot (Modular Robot) consists of a group of similar cells (Main part), each cell has five degrees of freedom. These cells are formed according to the shape and required function. The project involved: Cell design using Cero, force simulation using Ansys, mathematical model derivation , control circuit design and model fabrication.
